# Фірмові ассети КП

Покладіть сюди файли з проєкту «Генерація КП Стан Комплект» (Project knowledge)
і закомітьте разом із кодом:

| Файл                  | Призначення                                              |
| --------------------- | -------------------------------------------------------- |
| SK_cover_base.png     | Обкладинка (логотип, «КОМЕРЦІЙНА ПРОПОЗИЦІЯ», контакти)  |
| SK_footer.png         | Нижній колонтитул (зелено-помаранчева графіка)           |
| SK_company_top.png    | Верхній фотоколаж розділу «Про компанію»                 |
| SK_company_bottom.png | Нижній фотоколаж розділу «Про компанію»                  |

Без цих файлів генерація працює, але:
- обкладинка збирається у спрощеному текстовому вигляді;
- колонтитул і фотоколажі пропускаються;
- у результаті генерації з'являються відповідні попередження.

Позиція назви верстата та фото на обкладинці налаштовується константами
`COVER_TITLE` і `COVER_PHOTO_Y` у `src/lib/kp/constants.ts` (значення в DXA,
1 см ≈ 567 DXA). Якщо після першої генерації назва не потрапляє в сіру смугу
обкладинки — підженіть `y` і перегенеруйте.
